# Hackernews

Built with the following technology stack:

### UI

-   Next.js
-   Next Auth
-   TypeScript
-   React Query
-   Redux Toolkit
-   [NextUi](https://nextui.org/) component library
-   Tailwindcss
-   Styled Components
-   react-hook-form
-   zod

### API

-   TypeScript
-   trpc
-   Prisma ORM
-   MySQL DB
-   zod

### DevOps

-   Vercel serverless cloud deployment for application
-   Planetscale cloud deployment for database
